Descrição: Um site de banco digital que permite a criaçao de usuario, login, saque, deposito, tranferencia de dinheiro para outras contas e visualização da lista de usuarios com saldo e limite.
Tecnologia: VSCode, Supabase, HTML5, CSS3, JavaScript, IA.
Execução: Abra o arquivo index.html e insira os seguintes dados: nome, cpf, senha, email, e crie uma senha de no minimo 8 caracteres, voce pode esolher se quer fazer um deposito inicial e aumentar o limite
ao clicar em entrar, se todas as informaçoes estiverem preenchidas e validas voce sera guiado ate a tela principal, onde podera escolher entre fazer um deposito um saque aumentar o limite ou uma transferencia
se tentar sacar transferir mais do que tem de saldo, o site perguntara se voce quer usar do seu limite para realizar a ação
